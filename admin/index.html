Painel do Blog

Crie e edite os posts direto no GitHub — sem precisar de login extra. Os textos publicam sozinhos no site em poucos minutos.

✍️ Escrever novo post 📂 Ver / editar posts existentes
Como funciona: ao clicar em "Escrever novo post", o GitHub abre um arquivo já com o modelo pronto. É só preencher o título, o resumo e o texto, e clicar em Commit changes embaixo. O blog atualiza sozinho.
